# HR & Operations Performance Audit

This is demonstrates advanced proficiency in HR data visualization and the use of conditional logic to audit operational tasks.

## 📋 Project Overview
This project involved a dual-track analysis:
1. **Visualization:** Creating a 2D Clustered Column Chart to visualize employee performance and organizational metrics.
2. **Operational Audit:** Using conditional formulas to analyze task completion statuses and employee IDs across a centralized dataset.

## 🚀 Key Tasks Completed
- **Data Visualization:** Developed a Clustered Column Chart to compare multiple data series, ensuring clear visual communication of HR trends.
- **Task Auditing:** Processed the `taskcsv` dataset to identify and categorize task statuses (Completed vs. Pending).
- **Relational Verification:** Verified Employee IDs and their associated tasks to ensure data consistency across the audit.
- **Formatting & Design:** Applied professional layout standards to the final Excel report for executive-level presentation.

## 📂 Repository Contents
- `taskcsv.xlsx`: The core dataset containing employee task logs and status reports.
- `Assignment_11_Documentation.docx`: Detailed project requirements and logical steps.
- `Visuals/`: Includes the 2D Clustered Column charts generated during the analysis.

## 🛠️ Technical Skills
- **Excel Data Visualization:** 2D Clustered Column Charts.
- **Advanced Logic:** Handling relational data between tasks and employee records.
- **Business Reporting:** Converting raw CSV data into a structured assignment solve.
